# LAMMPS-MACE Water Diffusion

## Overview

This project investigates bulk water diffusion using molecular dynamics simulations in LAMMPS with MACE machine-learning interatomic potentials.

## Objectives

- Simulate bulk water dynamics
- Compute Mean Squared Displacement (MSD)
- Estimate diffusion coefficients
- Validate MACE model performance

## Software

- LAMMPS
- MACE
- Python
- NumPy
- Pandas
- Matplotlib
